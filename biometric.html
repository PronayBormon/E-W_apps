Introduction to biometric verification

Your wallet password will be encrypted and stored in the Keychain/ Keystore of your mobile device, biometrics (fringerprint or facial) will decrypt the password for transaction esecution.

Please backup the password immediately after enabling biometric verification feature. To reset the password, you can reimport the wallet using Mnemonic Phrase or Private Key.

Security reminder

Please check on the security level of your mobile device's biometric verification features for large assets transfer, please execute by manually inputting you password

Please refrain from enabling biometric verification shared mobile devices

Disclaimer

The security levels for biometric verification very among mobile device, users are recommended to proceed with caution. BRICS token will not be held liable for any risks or loss arising from vulnerabilities in the biometric verification feature.
